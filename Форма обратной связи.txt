1. На странице должен присутствовать заголовок первого уровня с id="title".
2. На странице должен присутствовать параграф с коротким объяснением  и id="description".
3. На странице должна быть форма с id="survey-form".
4. Внутри формы должно быть поле ввода имени с id="name".
5. Внутри формы должно быть поле ввода email с id="email".
6. Если внутри поля с email будет введен не нужного формата, то должна быть HTML5 валидация.
7. Внутри формы должно быть цифровое поле для ввода возраста id="number".
8. Если внутри цифрового поля вводить не цифру, то должна сработать HTML5 валидация.
9. Если внутри цифрового поля  числа вне диапазона ввода чисел,
    которые определены атрибутами min и max, я увижу ошибку проверки HTML5.
10. Для полей ввода имени, адреса электронной почты и номера внутри формы создать
    соответствующие метки, которые описывают назначение каждого поля со следующими идентификаторами:
    id="name-label", id="email-label", id="number-label".
11. Для полей ввода имени, адреса электронной почты и номера должен присутствовать
     пример для заполнения, который дает описание или инструкции для каждого поля.
12. Внутри формы должен быть выпадающий список с id="dropdown", с вариантами ответа, который можно выбрать.
13. Внутри элемента формы должно быть поле из одной или нескольких групп переключателей.
    Каждая группа должна быть сгруппирована с использованием атрибута name.
14. Внутри элемента формы должно быть несколько полей из серии флажков,
    каждое из которых должно иметь атрибут value.
15. Внутри формы должно быть поле для ввода коментария.
16. Внутри поля должна быть кнопка с id="submit"б которая отправляет форму
